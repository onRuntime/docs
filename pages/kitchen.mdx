# Kitchen

Kitchen is an open source UI kit for React and React Native. It provides a set of optimized, customizable and reusable styled-components that make it super easy to create websites and apps.

If you want to learn more about how to use Kitchen to build your projects, check out the [documentation](https://kitchen.tonightpass.com/). And if you'd like to contribute to the project, check out the [GitHub repository](https://kitchen.tonightpass.com/).

Please don't forget to read the [Contributing Guidelines](/contributing/introduction) before submitting your PRs.
